# P002 — Snake Classic

## Status
Prototype APK v0.7

## Goal
Create a basic classic Snake game for Android and test via direct APK download/sideload before deciding whether to publish on Google Play.

## Current Features
- Grid-based classic Snake gameplay
- Main menu and options menu
- Control options: swipe, on-screen buttons, tap-to-turn
- Difficulty/speed options: easy, normal, hard
- Theme/color options: neon, classic, ocean
- Layout options: compact and tall
- Sound effects toggle
- Ad placeholder toggle (real ads later require AdMob setup)
- Score display
- Local high score saved on device
- Ready, paused, and game-over screens

## Tech Stack
- Native Android
- Java custom `View`
- Gradle + Android Gradle Plugin
- Target SDK/API 35

## Next Steps
1. Install and play the debug APK on phone.
2. Report control/gameplay/design changes.
3. Decide whether to add menu, difficulty, sounds, ads, themes, or Play Store polish.
